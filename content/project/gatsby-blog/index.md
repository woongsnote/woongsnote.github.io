## 1. 개요

- **React** 기반의 **Gatsbyjs**를 활용한 블로그 제작.

## 2. 프로젝트 내용

### 2-1. **Gatsby로 만들게 된 이유**

React를 공부하다가, React 기반의 GatsbyJs에 대해 알게 되었습니다.

개발 블로그를 직접 만들기 위해 공부하던 상황이었기 때문에 GatsbyJs를 더 알아보았습니다.

그 결과, 라이브러리도 많고 참고할 자료도 많다고 생각되어 이를 기반으로 블로그를 개발하기로 했습니다.

### 2-2. **블로그 구조**

여러 블로그 스타터 등을 비교하다가, zoomcoding 프로젝트를 찾았습니다.
구현하고자 하는 기능 대부분을 보유하고 있어서, 이 테마를 기본으로 잡고 블로그를 개발했습니다.
